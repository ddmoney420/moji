# Changelog

All notable changes to Moji will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-30

### Added

#### Commands
- `/moji banner` - ASCII art text banners with 10 fonts
- `/moji fortune` - Random fortunes across 6 categories
- `/moji say` - Talking creatures with 9 creature options
- `/moji emoji` - Emoji search with 90+ developer-focused emojis
- `/moji commit` - Gitmoji reference for commit messages
- `/moji flex` - Fortune + creature combo
- `/moji divider` - 10 decorative divider styles
- `/moji box` - Text in decorative boxes
- `/moji header` - File/section headers
- `/moji rainbow` - Rainbow gradient text

#### Creatures
- cow (classic cowsay)
- tux (Linux penguin)
- ghost (minimal spooky)
- cat (cute kitty)
- robot (geometric minimal)
- owl (wise bird)
- dino (T-Rex energy)
- claude (custom Claude mascot)
- bunny (cute rabbit)

#### Fortunes
- 30 dev jokes and wisdom
- 20 Unix philosophy quotes
- 20 Zen proverbs
- 20 motivational quotes
- 25 absurdist humor
- 20 practical tips

#### Features
- 12 color gradients (rainbow, fire, ocean, forest, sunset, neon, ice, lava, matrix, synthwave, pastel, monochrome)
- 4 color themes (default, dracula, nord, monokai)
- CLAUDE.md integration examples

### Technical
- Claude Code skill-based architecture
- JSON data files for easy extension
- MIT licensed
