# Create Examples Directory with Shell Scripts and Usage Demos

## Problem Description

New users lack practical examples showing how to combine moji's features. An examples directory with shell scripts would demonstrate common use cases, effect chaining, and advanced workflows.

## Solution Approach

1. Create `examples/` directory at project root
2. Add shell scripts demonstrating key features
3. Include a README explaining each example
4. Cover beginner to advanced use cases
5. Make scripts executable and well-commented
